-------------------------------
INTRODUCTION
-------------------------------

Modifiers are useful to change the renderer behavior for display text coming from mtp chat servers.
For instance, take a look at "someonesays.lua" to imagine what you can do.

If you want to write or modify a modifier, create (or open) a file named "tokentype".lua into "modifiers" directory. Your modifications will be taken into account immediatly without restarting CeB. 
"tokentype" can be any string from the list below.

-------------------------------
FUNCTIONS
-------------------------------

You have several functions to modify the aspect of a mtp token :

* string get_session_info(string arg)

Returns an info concerning current session.
For example : get_session_info("login") will return "Fooman" if your nick is "Fooman"

* void session_send(string sentence)

Send a sentence to the server
For example : session_send("I love meat ice cream")

* int segments_count()

Return the segments count
For example: in "<Fooman> I love meat ice cream", segments_count will be 4 ("<", "Fooman", "> ", "I love meat ice cream")

* void set_segments_count(int arg)

Set the segments count
If arg is lower than the true segments count, last segments are removed
If arg is upper, new segments with a default font, a black color and empty text are added
This function can be useful to cancel the display of a render sequence, by using "set_segments_count(0)"

* string get_segment_text(int segmentNum)

Returns a segment text.
For example: in the above sentence, get_segment_text(2) will be "Fooman"

* void set_segment_text(int segmentNum, string text)

Replace the segment text at place <segmentNum> by <text>

* string get_segment_color(int segmentNum)

Returns a segment color in hexa form "RRGGBB"

* void set_segment_color(int segmentNum, string color)

Set a segment color in hexa form
